# SquareIt
##Application for Rendering Rectangels on an SVG.

>Want to see cool rectangles?! Enter width, height, and color to see them rendered on the svg canvas.

### Installing Dependencies

```sh
npm install
```

### Testing

```sh

```

### Build and Run

